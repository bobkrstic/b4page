Recipes Search

Searching for recipes app. Created using react.js and react hooks. Please sign up for edamam account which is free and will let you query the data from thier database about ten times per minute. Follow the link to the source code and the README.md file for instructions. For demonstration purposes, clone the repo, and npm install, and npm start. You will be able to use my own key to the edamam database.
